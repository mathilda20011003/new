# 🚀 飞书集成快速配置清单

> 打印或保存此清单，按顺序完成每一步

---

## ✅ 配置步骤

### 第一步：配置飞书机器人（10分钟）

- [ ] 1. 打开 https://botbuilder.feishu.cn/home/my-command
- [ ] 2. 点击"新建机器人指令"
- [ ] 3. 选择触发器 → "Webhook 触发"
- [ ] 4. **复制 Webhook 地址到记事本**（重要！）
- [ ] 5. 配置参数（复制下面的 JSON）：

```json
{
  "message_type": "text",
  "content": {
    "total_titles": "{{内容}}",
    "timestamp": "{{内容}}",
    "report_type": "{{内容}}",
    "text": "{{内容}}"
  }
}
```

- [ ] 6. 点击"完成"
- [ ] 7. 选择操作 → "通过官方机器人发消息"
- [ ] 8. 消息标题填写：`AIGC动态漫热点监控`
- [ ] 9. 添加消息内容字段（按顺序）：
  - `report_type`
  - `text`
  - `total_titles`
  - `timestamp`
- [ ] 10. 保存并发布
- [ ] 11. 手机飞书搜索"机器人"并开启应用

---

### 第二步：配置 GitHub（5分钟）

- [ ] 1. 打开 https://github.com/mathilda20011003/new
- [ ] 2. 点击右上角 "Fork" 按钮
- [ ] 3. 进入你 Fork 后的项目
- [ ] 4. 点击 Settings → Secrets and variables → Actions
- [ ] 5. 点击 "New repository secret"
- [ ] 6. Name 填写：`FEISHU_WEBHOOK_URL`
- [ ] 7. Secret 填写：你保存的 Webhook 地址
- [ ] 8. 点击 "Add secret"

---

### 第三步：测试运行（2分钟）

- [ ] 1. 点击项目顶部的 "Actions" 标签
- [ ] 2. 如果看到提示，点击 "I understand..." 启用
- [ ] 3. 左侧选择 "Hot News Crawler"
- [ ] 4. 点击 "Run workflow" → 再次点击确认
- [ ] 5. 等待 1-2 分钟
- [ ] 6. 检查飞书是否收到消息

## 🔧 常用调整

### 修改关键词
1. 进入项目 → `config` → `frequency_words.txt`
2. 点击编辑按钮 ✏️
3. 修改后点击 "Commit changes"

### 调整推送时间
编辑 `config/config.yaml`：
```yaml
notification:
  push_window:
    enabled: true      # 启用时间窗口
    time_range:
      start: "09:00"   # 开始时间
      end: "18:00"     # 结束时间
```

### 查看运行日志
1. 项目 → Actions
2. 点击最新的运行记录
3. 查看详细日志

## 🆘 遇到问题？

### 飞书收不到消息
1. 检查 Webhook 地址是否正确
2. 检查飞书机器人是否已发布
3. 检查手机飞书是否开启"机器人"应用
4. 查看 GitHub Actions 运行日志

### 新闻太少
- 减少过滤词（!开头的）
- 增加更多关键词

### 新闻太多
- 增加过滤词（!开头的）
- 使用必须词（+开头的）限定范围
