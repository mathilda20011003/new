# 微信公众号RSS爬虫配置说明

## 📁 配置文件结构

```
.env                          # 🔑 统一配置文件（所有密钥都在这里）
config/wechat_accounts.yaml   # 📋 公众号列表配置
```

## 🔑 环境变量配置 (.env)

所有敏感信息和基础配置都统一在 `.env` 文件中：

```bash
# 飞书推送
FEISHU_WEBHOOK_URL=您的飞书Webhook地址

# OpenRouter API
OPENROUTER_API_KEY=您的OpenRouter API密钥

# WeWe RSS服务
WEWE_RSS_BASE_URL=https://ssys2025.zeabur.app/

# AI配置
AI_PROVIDER=openrouter
AI_MODEL=google/gemini-2.5-flash-lite-preview-09-2025
AI_MAX_TOKENS=150
```

## 📋 公众号配置 (config/wechat_accounts.yaml)

只包含公众号列表，不包含敏感信息：

```yaml
accounts:
  - name: "短剧自习室"
    feed_id: "MP_WXS_3906677264.atom"
    keywords:
      - "漫剧"
      - "AI漫剧"
```

## 🚀 运行方式

```bash
python run_wechat_rss.py
```

## ✅ 优势

1. **安全性** - 所有密钥统一管理，不会意外提交到Git
2. **简洁性** - 配置清晰，易于维护
3. **灵活性** - 环境变量优先级高于配置文件
